# 🎯 Vibe Coding Workshop - Lab Exercises

An interactive HTML application for the **Vibe Coding Workshop** featuring 6 hands-on lab exercises that demonstrate GitHub Copilot's capabilities in web development.

## 🌐 Live Demo

Visit the workshop: [https://tantunes_cisco.github.io/vibe-coding-workshop/](https://tantunes_cisco.github.io/vibe-coding-workshop/)

## 📋 Workshop Labs

Each lab takes approximately **~20 minutes** and runs entirely in the web browser:

1. **✅ Personal Habit Tracker** - Productivity Tool
2. **⭕ Tic-Tac-Toe Game** - Browser Game with AI
3. **🃏 Emoji Memory Match Game** - Memory Card Game
4. **🧩 Sudoku Game** - Grid-based Puzzle
5. **❓ Interactive Quiz Builder** - Learning Tool
6. **📋 Mini Kanban Board** - Task Management

## 🚀 Features

- **Interactive Lab Selection** - Click any lab to view detailed instructions
- **Copilot Prompts** - Pre-written prompts to help participants
- **Learning Outcomes** - Clear objectives for each lab
- **Responsive Design** - Works on desktop and mobile
- **Modern UI** - Clean, gradient design with smooth animations

## 📦 Usage

Simply open `index.html` in any modern web browser. No installation or dependencies required!

## 🎓 Workshop Details

- **Target Audience:** All levels (technical and non-technical)
- **Technologies:** HTML, CSS, JavaScript
- **Duration:** ~20 minutes per lab
- **Focus:** GitHub Copilot-powered development
